EXAMBROWSER APPLICATION SYSTEMS BERBASIS ANDROID NATIVE

Merancang dan membangun aplikasi ExamBrowser tangguh berskala native mobile architecture menggunakan Android Studio (Java/Kotlin).

Solusi ini di-deploy khusus sebagai infrastruktur penunjang pelaksanaan ujian semester siswa berbasis smartphone.

Fitur Keamanan Utama:
  • Screen-Pinning Engine - Kiosk Mode (mengunci layar penuh)
  • Anti-Screen Capture Protection - Memblokir screenshot & screen recording
  • Smart Link Validation - Render halaman CBT di sandbox environment
  • Hardware Control - Menonaktifkan tombol Home, Recent Apps, dll
  • Active Deployment - Masih digunakan sampai sekarang di sekolah

SISTEM INFORMASI BANK MINI IQTI BERBASIS ANDROID

Membangun sistem informasi manajemen nasabah berbasis Android yang dirancang untuk menyajikan informasi saldo secara real-time, push notification transaksi, detail informasi akun, serta fitur transfer antar-nasabah secara instan.

Sistem ini mengintegrasikan panel admin berbasis web menggunakan CodeIgniter (PHP Framework), jQuery, dan MySQL sebagai basis data terpusat, serta ditenagai oleh Ionic Framework pada sisi aplikasi Android.

Sistem mencakup fungsionalitas utama seperti pendaftaran nasabah baru, pencatatan transaksi menabung (debit/kredit), hingga pembuatan laporan keuangan secara otomatis.

Catatan Proyek:

Proyek ini merupakan aplikasi pertama yang saya rancang dan sukses dideploy sebagai Tugas Akhir (TA) pada tahun 2019. Saat ini, masa operasional dan akses ke server demonstrasi sistem telah resmi ditutup.

SISTEM INFORMASI PPDB 2020

Software ini telah digunakan di SMK Maarif NU Tirto dan berjalan lancar hingga saat ini.

Aplikasi ini mencakup proses pendaftaran siswa baru, pengecekan data siswa, pengaturan jadwal pendaftaran, serta manajemen hak akses admin dan superadmin.

Pengembangan & Optimalisasi Fitur Custom:
  • Perbaikan arsitektur dan core bug sistem autentikasi logout.
  • Integrasi visualisasi analitik grafik statistika kompetensi keahlian yang diminati.
  • Refaktor tabel database relasional (perubahan Nama Panggilan menjadi NIK).
  • Sinkronisasi integrasi schema database dengan sistem pelaporan dan ekspor data siswa.
  • Penyediaan modul cetak formulir pendaftaran fisik dan bukti pendaftaran digital.
  • Implementasi UI micro-interaction menggunakan SweetAlert library.

Catatan:

Aplikasi ini dikembangkan oleh R. Anggit Herdianto. Anda dapat melihat repository asli di: github.com/AnggitHerdianto
