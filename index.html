EXAMBROWSER APPLICATION SYSTEMS BERBASIS ANDROID NATIVE

Merancang dan membangun aplikasi **ExamBrowser** tangguh berskala *native mobile architecture* menggunakan lingkungan **Android Studio (Java/Kotlin)**. Solusi ini di-deploy khusus sebagai infrastruktur penunjang pelaksanaan agenda evaluasi belajar (Ulangan Semester) siswa berbasis perangkat *smartphone/handphone*.

Aplikasi ini mengintegrasikan protokol manajemen keamanan terpusat untuk meminimalisir potensi kecurangan akademis secara presisi selama ujian berlangsung.

Fitur Keamanan Korporat & Keunggulan Sistem:

  • **Screen-Pinning Engine:** Mengunci layar secara penuh (*Kiosk Mode*) untuk mencegah siswa berpindah ke aplikasi lain, membuka peramban web luar, ataupun membuka catatan dokumen.
  • **Anti-Screen Capture Protection:** Memblokir seluruh fungsionalitas perintah pintas tangkapan layar (*screenshot*) dan perekaman layar (*screen recording*) melalui kontrol level sistem internal.
  • **Smart Link Validation & Render Engine:** Memuat halaman server Computer Based Test (CBT) sekolah secara responsif, stabil, dan terisolasi penuh di dalam *sandbox environment*.
  • **Hardware Control Management:** Menonaktifkan akses tombol navigasi bawaan perangkat keras (seperti tombol *Home*, *Recent Apps*, dan fungsi pembagian layar/*split screen*) selama sesi pengerjaan ujian aktif.
  • **Active Deployment Status:** Hingga saat ini, sistem aplikasi ini masih aktif diimplementasikan secara berkelanjutan di lingkungan internal sekolah untuk menjamin integritas pelaksanaan evaluasi belajar di setiap semester.

SISTEM INFORMASI BANK MINI IQTI BERBASIS ANDROID

Membangun sistem informasi manajemen nasabah berbasis Android yang dirancang untuk menyajikan informasi saldo secara real-time, push notification transaksi, detail informasi akun, serta fitur transfer antar-nasabah secara instan.

Sistem ini mengintegrasikan panel admin berbasis web menggunakan CodeIgniter (PHP Framework), jQuery, dan MySQL sebagai basis data terpusat, serta ditenagai oleh Ionic Framework pada sisi aplikasi Android-nya. Sistem mencakup fungsionalitas utama seperti pendaftaran nasabah baru, pencatatan transaksi menabung (debit/kredit), hingga pembuatan laporan keuangan secara otomatis.

*Proyek ini merupakan aplikasi pertama yang saya rancang dan sukses dideploy sebagai Tugas Akhir (TA) pada tahun 2019. Saat ini, masa operasional dan akses ke server demonstrasi sistem telah resmi ditutup.

Sistem informasi PPDB 2020

Software ini telah digunakan di SMK Maarif NU Tirto dan berjalan lancar. pembuat aplikasi ini ialah R. Anggit Herdianto anda bisa mendownload nya di: http://github.com/AnggitHerdianto. Aplikasi ini melingkupi pendaftaran siswa baru, pengecekan data siswa, pengaturan jadwal pendaftaran, serta manajemen hak akses admin & superadmin.

Pengembangan & Optimalisasi Fitur Custom:

  • Arsitektur & perbaikan core bug sistem autentikasi logout.
  • Integrasi visualisasi analitik grafik statistika kompetensi keahlian diminati.
  • Refaktor tabel database relasional (Nama Panggilan menjadi NIK).
  • Sinkronisasi integrasi schema database ke sistem pelaporan ekspor data siswa.
  • Penyediaan modul cetak formulir pendaftaran fisik & cetak bukti pendaftaran digital.
  • Implementasi UI micro-interaction engine menggunakan SweetAlert library.
